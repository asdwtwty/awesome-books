## 1. 三种维度建模范式简介

按照 [Curcio et al., 2021《Seven Mathematical Models of Hemorrhagic Shock》](https://github.com/asdwtwty/0D-1D-3D) 第 2.2 节 "Dimensionality" 与 [Shi/Lawford/Hose 2011《Review of Zero-D and 1-D Models》](https://github.com/asdwtwty/0D-1D-3D) 的分类：

| 维度 | 数学结构 | 主要可解析量 | 计算成本 | 典型用途 |
|---|---|---|---|---|
| **0D (集中参数 / Lumped Parameter)** | ODE / DAE，电-液类比 (R, C, L)，各腔室内压、流、容均匀 | 全身腔室间压力、流量、容量、心输出量、HR、SVR、SpO₂、压力反射 | 极低，可实时 | 全身循环、自主神经/体液调节、器官-器官交互、闭环控制器评估 |
| **1D (分布式参数 / 一维波传播)** | 沿血管轴线的双曲型 PDE (连续性 + 动量)，配 0D 末端边界 (Windkessel) | 沿动脉树的脉搏波传播、PWV、波反射、局部 P(x,t)、Q(x,t) | 中等 | 解剖详细的动脉/静脉网络、脉搏波分析、波反射诊断 |
| **3D (CFD / FE)** | Navier-Stokes 全场求解，常配壁面流固耦合 (FSI) | 三维速度场、WSS、再循环区、复杂分叉/狭窄局部流 | 高，难以实时 | 局部病变 (狭窄、动脉瘤、瓣膜)、血栓预测、装置设计 |

文献中越来越多采用 **多尺度耦合 (multiscale)**：3D 局部 ⇄ 1D 网络 ⇄ 0D 全身闭环。

## 3. 引文 (References) 中常被引用的核心 0D HS 模型族

通过解析 `0.Seven Mathematical Models of Hemorrhagic Shock.pdf` 的 190 余条参考文献，可以梳理出**专门用于失血性休克** 的 0D 经典模型谱系：

| 模型族 | 关键文献（在 Curcio 综述里的索引） | 可仿真 HS 的特性 |
|---|---|---|
| **Windkessel 2/3/4-element** | Frank 1899；Westerhof 1971；Stergiopulos 1999 | 仅作为"末端边界"，本身不能仿真 HS |
| **Guyton 系列 (G72/G84)** | Guyton 1972 [141]; Montani 1989 [153]；Hester/HumMod | 全身长期调控，含肾、压力感受器、ADH/RAAS——可仿真长时程 HS |
| **Grodins 1959** | Grodins [1]，Cheng/Albanese 系列 | 心-肺联合，4 腔室，是 21 号论文 (战创伤 CR 模型) 的母版 |
| **Siam et al. 2003** | Siam, Kadan, Flaishon, Barnea [158] | 专用于评估出血状态下不同晶/胶体复苏的氧供 |
| **Batzel et al.** | Batzel-Kappel-Timischl [172,173] | 显式聚焦失血诱发的控制系统响应 |
| **Beard biosimulation 框架** | Beard 2012 [184]；JSim | 整合主动脉力学 + 调控 |
| **Zenker 2007** | Zenker, Rubin, Clermont [13] | 简洁高保真，HS + 复苏专用，最常被引用 |
| **HumMod / BioGears / Pulse / Muse** | Hester 等 — 商业/开源全身仿真器 (论文 #1 比较) | 现成出血场景，但与外部接口受限 |
| **Vodovotz / Brown / Cannon 炎症-凝血 ODE 链** | [#2, #3, #01] | 把 HS 与免疫炎症耦合，能预测 MODS/死亡率 |
| **Hahn / Bighamian / Kramer 集中参数 + 复苏控制** | A Lumped-Parameter ... Automated Fluid Resuscitation；#11 | 专为闭环复苏设备 (PCLC) 评估 |

## 4. 哪些模型适合"创伤失血性休克 (T/HS) 血流动力学建模"？

**T/HS 建模的关键需求**：

1. **全身闭环循环 + 血容量动力学**——必须能描述出血导致的血容量、SV、CO、MAP、HR 的动态级联。
2. **压力感受器 / 自主神经反射 + 体液调节 (RAAS/ADH)**——决定休克代偿与失代偿。
3. **跨毛细血管液体重分配 (transcapillary refill)**——必须有 *组织间隙腔室*，否则无法模拟自身血液稀释及晶/胶体复苏。
4. **多场景输入**：出血率、晶体/胶体/血制品输注率、血管活性药物。
5. **可与炎症/凝血/MODS/死亡时间 outcome 链路耦合**（高级需求）。
6. **计算实时 / 可大规模虚拟队列**——用于训练 AI、闭环控制器、个体化决策。

#### 4.1 推荐 — 强适用 (★★★)

| 优先级 | 文献 | 维度 | 推荐理由 |
|---|---|---|---|
| **P1** | **Zenker 模型** (Curcio 综述 §4.7) | 0D | 业内事实上的 HS "标准小模型"，含心室 ESPVR/EDPVR、压力反射、出血-复苏，参数少、易识别、文献引用最多 |
| **P1** | **Jin & Reifman 2023 — CR model for battlefield trauma** (#21) | 0D | 74 个 ODE，已包含**间质液腔室**，可同时仿真出血、晶体复苏、气道损伤；27 受试者 5 项研究跨场景验证；DoD 项目，文档详尽 |
| **P1** | **Chalumuri & Bighamian 2024 — Lumped-Parameter for Automated Fluid Resuscitation** | 0D | FDA OSEL 主导，输入直接是出血率/输注率，输出即 Hct、HR、SV、CO、MAP；27 羊 + 12 猪两数据集验证；面向 PCLC 闭环控制器评估 |
| **P1** | **Cannon, Vodovotz et al. 2024 — Digital twin for HS resuscitation** (#01) | 0D 三腔室 ODE | 直接将病理生理 + 炎症 + 凝血与 PROMMTT 临床数据耦合，能输出"死亡时间"和不同复苏方案 (晶体 vs 血浆 vs 血浆+RBC) 的对比，适合做复苏策略推荐 |
| **P2** | **Curcio 综述里的 Siam/Batzel/Guyton 模型** | 0D | 经典 HS 模型；如果只关心"宏观循环 + 反射"，结构最简、解释性最好 |
| **P2** | **Brown, Vodovotz 2015 — Trauma in silico** (#3) | 0D | 多腔室 ODE + 1 万虚拟人群 + IL-6 SNP，适合做 *in silico* 临床试验 / 死亡率预测 |
| **P2** | **Kao & Hahn 2023 — Vasoplegic Shock + Vasopressor** (#11) | 0D | 如果方案需要在失血性休克晚期/休克后阶段加入血管活性药 (PHP) 闭环控制 |

#### 4.2 中等适用 (★★)

| 文献 | 维度 | 适用情境 |
|---|---|---|
| **Barnes 2020 — BioGears/HumMod/Muse 比较** (#1) | 0D 整机 | 可直接拿现成软件做出血场景仿真，但接口受限 |
| **Lishak 2023 — Variable HR CV-Resp** (#31) | 0D | 需要 HRV、长时仿真时的方法学增强 |
| **Nieman 2012 — Two-Compartment Endotoxin in Swine** (#2) | 0D | T/HS 后期常并发脓毒症，可作为炎症建模分支 |
| **Garber 2022 — Critical Role of LPM (#02)** + **Shi 2011 (#16)** | 综述 | 选型 / 写综述 / 多尺度耦合时的方法学参考 |
| **Blanco ADAN / Müller ADAVN / Dalmaso ADAVN+lung** | 1D | 需要研究 *出血时脉搏波形态变化、PWV、外周波反射* 时；或作为 0D 全身模型的"动脉树边界条件" |
| **openBF, first_blood (Wéber)** | 1D | 提供高效 1D 求解器与虚拟患者数据库，可批量产生失血场景的脉搏波 |

#### 4.3 不适用 / 仅作为局部插件 (☆)

- **Xie 2018 (#32) / Mirramezani 2019 (#33)** — 冠脉狭窄局部 CFD/ROM，与 T/HS 全身血流动力学几乎无关。  
- **Shavik 2020 (#34)** — PAH 心室-肺动脉 3D-FE，主要解决肺动脉高压。  
- **Romero 2025 (VCS shape model)** — 形状/网格基础设施，与 HS 无直接关系。  
- **Golse 2022 (post-hepatectomy)** — 0D 模型，但目标病种是肝切除门脉高压。  
- **Rubio/Marsden 2024 (bifurcation ROM)** — 冠脉树 ROM 工程，不针对 HS。

## 5. 推荐的研究路径建议

针对**创伤失血性休克的血流动力学建模**，建议按以下三层堆栈搭建：

```
┌─────────────────────────────────────────────────────────────┐
│  L3  outcome / 决策层                                          │
│  · Vodovotz 链 (#01, #3, #2)：炎症 + 凝血 + MODS + 死亡时间预测 │
└─────────────────────────────────────────────────────────────┘
                           ▲
┌─────────────────────────────────────────────────────────────┐
│  L2  全身血流动力学闭环 (核心，0D)                                │
│  · Zenker (Curcio §4.7)  — 极简 HS 基线                       │
│  · Jin & Reifman 2023 CR (#21) — 心+肺+间质液+战创伤验证        │
│  · Chalumuri-Bighamian 2024 — 出血+复苏+羊/猪验证 (PCLC)        │
│  · 可加 Kao-Hahn 2023 vasopressor 子模块 (#11)                 │
└─────────────────────────────────────────────────────────────┘
                           ▲
┌─────────────────────────────────────────────────────────────┐
│  L1  局部脉搏波 / 大动脉 (按需，1D)                              │
│  · ADAN / ADAVN (Blanco, Müller) 作为 0D 末端 → 1D 边界          │
│  · openBF / first_blood 求解                                  │
└─────────────────────────────────────────────────────────────┘
   (3D CFD 一般不直接用于 HS 全身建模，仅在装置设计/局部病变时作为多尺度耦合的最末端)
```

**实施建议**：
1. **若目标是临床决策支持 / 闭环复苏设备**：直接采用 Chalumuri-Bighamian (2024) 或 Jin-Reifman (2023) 这类已用动物-人数据交叉验证的 0D 模型，并参照 #16、#02 综述补充压力反射与肾-体液调节模块。
2. **若目标是机制研究 / *in silico* 试验**：采用 Cannon-Vodovotz (2024) + Brown (2015) 路线，把 0D 血流动力学与炎症-凝血 ODE 链路耦合，并基于 PROMMTT 等真实创伤队列做参数辨识。
3. **若需要分析失血时的动脉波形 / PWV 变化**：在 0D 全身模型外挂 ADAN + openBF；不必上 3D。
4. **如需大规模虚拟患者**：Wéber 2026 (`first_blood`) 与 Brown 2015 的 10 000 例虚拟人群是已有蓝本。
