#### Q2 — VVUQ 工作包缺失（强烈建议新增 RC4）

仅靠 RC3 末尾"不确定度量化"一句话不足以支撑数字孪生临床可信性。建议新增 RC4：

- **代码验证（Verification）**：与解析 Windkessel 解、Pulse Engine、Cannon 2024 ODE 公开代码、openBF 比对；
- **模型确认（Validation）**：与公开队列（PROMMTT、MIMIC-IV、PhysioNet）+ 合作医院回顾性创伤队列比对，给出 RMSE / AUROC / 校准曲线；
- **不确定性量化（UQ）**：参数 + 数据 + 模型三类不确定度的量化与传播，给出 95% 预测区间覆盖率、ECE 等指标。

### 2.5 研究方案与技术路线

#### Q5 — 0D-1D-3D 切换语义不清

文中同时出现"耦合""切换""动态切换分辨率"——多分辨率耦合（不同维度共存）和保真度切换（同阶段只用其中一种）是两种不同的技术路线。审稿人会追问：

- 切换的触发条件是什么？（事件 / 时间 / 误差驱动）
- 切换时边界条件如何传递？是否有保守性损失？
- 是否有在线降阶（Online ROM）以实现 1D/3D → 0D 的实时近似？

**建议改写为三层架构（详见修订版 §3.1）：**

1. **0D 全身闭环（实时引擎）**：始终在线，毫秒级；
2. **1D 动脉树（事件触发 / 残差驱动）**：止血带松紧、复苏液体推注、监护波形 vs 0D 残差超阈值时激活；
3. **3D 局部出血源（离线高保真 + 在线降阶代理）**：术前/伤情评估时基于 CT/超声重建，离线 CFD 解出关键流场参数，在线以 Kriging / POD-Galerkin 降阶代理模型返回 0D。
4. **接口处理**：参考 Rubio & Marsden 2024 的数据驱动 RRI 分叉模型解决 0D 与 1D/3D 衔接的精度损失。

#### Q4 — UKF 高维参数估计可行性

新版直接写"通过无迹卡尔曼滤波（UKF）实现状态-参数联合估计"，但循环-呼吸-神经-内分泌耦合模型常含 100+ 参数，UKF 直接联合估计存在协方差退化、σ 点维数爆炸、可辨识性差等问题。

**建议改为两阶段流程（详见修订版 §3.3）：**

1. **离线敏感性 + 可辨识性分析**：Sobol 全局敏感性指数对 100+ 参数排序，按一阶/总指数选 Top-K（K≈8–15）影响 MAP/HR/ETCO₂/SpO₂ 最显著的可辨识参数子集；
2. **在线 UKF 状态-参数联合估计**：仅对 Top-K 做联合估计，使用正则化协方差更新（Joseph form）保证正定，约束化 σ 点投影把参数限制在生理可行区间；
3. **不确定性量化**：UKF 输出基础上叠加 EnKF / 条件粒子滤波（CPF）做交叉校验，给出 95% 置信带；
4. **风险-收益可视化**：复苏方案 A/B 在 30/60/120 min 后伤情曲线 + 不确定带。

#### Q3 — 数据来源与伦理合规

新版仍未交代：

- 训练 PINN/Neural ODE 残差学习需要的标注数据从哪里来？
- 是否使用合作医院电子病历？是否走 IRB 伦理审批？
- 是否使用 PROMMTT/MIMIC-IV 公开队列？许可证如何？

**建议在 §3.5 明确写出（详见修订版）：**

1. **公开数据**：MIMIC-IV、PROMMTT、PhysioNet 急救数据集（遵循 DUA 与 PhysioNet credentialing）；
2. **合成数据**：基于 Pulse Engine、Jin & Reifman CR 模型生成 ≥1 万例虚拟伤员；
3. **合作医院数据**：与解放军总医院创伤救治中心 / 新疆医科大学附属医院 / 国家创伤医学中心等合作（拟），经 IRB 审批的回顾性脱敏创伤数据；
