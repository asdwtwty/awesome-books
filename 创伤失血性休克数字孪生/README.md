## 主要修订要点

1. **摘要与关键词重写**：去除"三位一体"等口号，统一全球流行病学口径，关键词替换为"院前急救"。
2. **国际背景增补**：引入 DARPA VITAL（DARPA-PS-26-26, 2026）作为国际同步性证据；引入 Sel 2024 JAHA 的 V&V&UQ 框架。
3. **研究现状-差距-应对对照表**（新增 §1.2.4）：与 Cannon 2024、Brown 2015、Jin 2023、Tivay 2022、Sel 2024、Rubio 2024 形成清晰边界。
4. **关键科学问题与研究内容显式映射**（§2.3）：3 个 KSP 与 4 个 RC 一一对应并附量化指标。
5. **0D-1D-3D 三层架构与切换准则**（§3.1）：明确事件触发、残差驱动、数据驱动 RRI 分叉补偿。
6. **PINN/Neural ODE 训练细节**（§3.2）：可微 ODE 求解器、联合损失函数、可解释性后验校验、训练数据三层来源。
7. **两阶段（Sobol → UKF → CPF）参数辨识 + UQ**（§3.3）：解决高维参数 UKF 不可辨识性问题。
8. **VVUQ 工作包**（§3.4，新增）：代码验证 + 模型确认 + 不确定性量化三层指标。
9. **数据来源与伦理合规**（§3.5，新增）：合成 + 公开 + 合作医院 + 健康志愿者四层数据策略。
10. **特色与创新改写**（§4）：从口号化改为"问题→对策→可量化效果"+ 与 4 篇代表作差异表。
11. **三年阶段化年度计划与里程碑**（§5.1）：按季度给出可交付物。
12. **考核指标量化**（§5.2）：增加 RMSE/AUROC/单步推理时延等定量指标。
13. **风险与应对**（§6，新增）：6 项风险与对策。
14. **参考文献**（§7）：统一格式、补充 Heldt 2002、Tivay 2022、Sel 2024、Rubio 2024、Brown 2015、DARPA VITAL 2026、突发事件创伤救治规范 2025 等条目。
