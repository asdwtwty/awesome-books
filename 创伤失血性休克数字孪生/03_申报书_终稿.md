## 摘要（限 400 字）

创伤是 45 岁以下人群首位死因，其中创伤失血性休克（THS）占创伤死亡的 30%–40%，是"可预防性死亡"的首要原因；从大出血至死亡中位时间约 2 小时，院前精准识别与个体化干预直接决定预后。然而突发事件院前场景下，监测维度受限、伤情演变剧烈、个体差异显著，传统经验救治难以兼顾。本项目针对多尺度血流动力学建模、机理-数据融合多系统建模、稀疏体征下的在线个性化与不确定性量化三大科学问题，研究机理-数据双驱动的伤员数字孪生关键技术，融合便携式传感器实时生命体征与创伤医学知识：(1) 0D-1D-3D 多分辨率可切换心血管模型，量化大出血扰动下全身-局部血流动力学并平衡保真度与实时性；(2) 物理信息神经网络与机理 ODE 联合训练的循环-呼吸-神经-内分泌耦合模型，刻画神经反射、内分泌调节等残差动力学；(3) 基于 Sobol 敏感性筛选与无迹卡尔曼滤波（UKF）的两阶段在线参数辨识，并配套验证-确认-不确定性量化（VVUQ）方案。形成具有自主知识产权的 THS 数字孪生伤员原型系统（0D 单步推理 ≤1 s），为事故灾害、反恐维稳等突发事件提供风险-收益可视化决策支持。

# 一、立项依据与研究内容

## 1. 项目的立项依据

### 1.1 研究背景与意义

创伤是全球 45 岁以下人群的首位死亡原因，全球疾病负担研究（GBD 2019/2021）显示每年因伤害死亡约 410–510 万人，其中道路交通伤约 130 万人。创伤失血性休克（Traumatic Hemorrhagic Shock, THS）占创伤死亡的 30%–40%，是严重创伤后"可预防性死亡"首要原因。对可见外出血，可通过局部压迫、止血带、药物或结扎方式及时救治；但腹腔/盆腔/胸腔等隐匿性出血伤员在早期生命体征代偿期常被低估，短时间内出现器官衰竭甚至死亡。我国每年因交通事故、工业事故、自然灾害和暴力伤害造成的严重创伤伤员超过 70 万人次。

THS 救治面临**时间窗紧、个体差异大、信息不完整**的三重挑战：(1) 从休克发生到死亡中位时间约 2 小时，"黄金一小时"内能否准确识别失血严重程度直接决定预后；(2) 地震、爆炸、群体伤亡事件中伤员陡增、救治力量稀缺、后送通道受阻，常规医疗模式难以兼顾快速失血评估、容量复苏决策、止血干预选择和后送优先级判定；(3) 伤员年龄、基础疾病、出血部位与速率对预后影响差异大，需要个性化复苏方案。同时，院前场景下监测手段受限，仅能采集部分生命体征，难以反映组织灌注、氧代谢、凝血等深层病理生理状态，难以前瞻预测伤情演变。

国家"十五五"规划纲要明确"健全紧急医学救援与院前急救体系，推进健康数智化建设"；国家卫生健康委要求"建立统一规范的院前-院内创伤分级预警机制、救治流程与信息共享"，《突发事件创伤伤员医疗救治规范（2025 年版）》进一步强调智能辅助救治系统的研究。新疆等地复杂的自然与社会环境对智能化突发创伤救治手段需求更为紧迫。

**国际同步性证据：** 美国 DARPA 生物技术办公室于 2026 年发布《Virtual-Integrated Twin for Autonomous Lifesaving》（VITAL, DARPA-PS-26-26）程序公告，明确将"持续更新的心血管数字孪生 + 患者数据 + 生物物理"作为面向急、慢、危重病例决策支持的国家级研究方向。美国国家科学院与 JAHA 综述也将"机理-数据混合建模"列为心血管数字孪生迈向临床的关键路径。本项目方向与上述国际前沿同步，且填补 THS 这一急救场景下数字孪生应用的研究空白。

数字孪生（Digital Twin, DT）通过在物理伤员与孪生模型之间建立"实时映射—预测演化—反馈优化"闭环，为破解上述三重挑战提供了从经验驱动转向模型与数据双驱动的可行路径。在数字空间中为每位伤员构建虚拟生理副本，实时跟踪伤情、预测演化、优化干预策略，对降低创伤可预防性死亡率、提升突发事件医疗救援能力具有重要研究价值与应用前景。

### 1.2 国内外研究现状

#### 1.2.1 创伤失血性休克评估

THS 是创伤致大量失血所致有效循环血量减少、组织灌注不足、细胞代谢紊乱与器官功能受损的临床症候群，常合并低血压（收缩压 < 90 mmHg、脉压 < 30 mmHg，或既往高血压者收缩压自基线下降 ≥40 mmHg）。临床多用休克指数（Shock Index, SI = 心率/收缩压，HR/SBP）粗略评估失血量与休克分级。SI 计算简单、可快速获取，但受心血管储备、血管顺应性、年龄、基础疾病影响存在个体差异。

创伤救治强调"黄金一小时"理念与 CABCDE 评估（出血控制、气道、呼吸、循环、神经功能、暴露与环境）。2025 年欧洲重症医学会（ESICM）循环性休克与血液动力学监测指南提出尿量减少、意识改变、皮肤灌注异常等床旁体征用于诊断和分级，并强调"连续、动态、多模态"监测对调整救治方案的关键作用。

随着创伤大数据建设进展，机器学习被用于关键风险指标筛选与休克预测预警。XGBoost + 序列前向搜索筛选关键风险指标；注意力增强网络与特征融合预测 THS；可穿戴设备的无创风险评分工具等已有报道。但上述方法依赖特定数据集训练，缺乏与生理机制的耦合，存在可解释性弱、外推不稳健、对医疗器械稀缺场景失效等问题。

#### 1.2.2 心血管系统数学模型

集总参数（Lumped-Parameter, 0D）方法类比电路，将血流、阻力、压力分别对应电流、电阻、电压。0D 模型从最简的两元件 Windkessel 模型（R-C）到含自主神经/内分泌反馈的复杂模型（如 Heldt 闭环模型、Guyton 模型）；Pulse 全身生理引擎也是基于 0D 描述循环、呼吸、代谢，可用于失血与复苏仿真，但对低血容量极端场景的预测存在系统性偏差。

为表现真实物理特性，需应用连续性方程、泊肃叶定律和纳维-斯托克斯方程实现质量与动量守恒。1D 模型增加长度方向空间分布，可有效表征动脉树脉搏波传播；3D 流固耦合可精细刻画特定血管段剪切力分布，但计算开销巨大，难以满足实时救治需求。Rubio & Marsden 2024 提出"数据驱动分叉处理 RRI 模型"嵌入 0D 求解器，将入口压力误差从 45% 降至 17%，为 0D-1D-3D 衔接提供了新近思路。

机理模型可提供高保真模拟，但因病理机制理解不完善与个性化数据稀缺受限。**目前缺乏面向 THS 急救场景、兼具高保真与实时性、可在线个性化的多分辨率（0D-1D-3D）血流动力学融合建模框架。**

#### 1.2.3 心血管数字孪生与融合建模方法

美国国家科学院《数字孪生的基础研究缺口与未来方向》指出：医疗数字孪生是对患者部分或全部生理系统的计算模型，与真实患者通过双向连接持续交互，定期用患者数据重新校准并提供个体化预测。当前医疗 DT 仍处探索期，建模路径分三类：

**(A) 机理为主：** Heldt 闭环模型、Pulse 引擎、Jin & Reifman 战创伤心-呼吸（CR）模型等，基于物理与生理方程构建低-中维 ODE/PDE。机制模型生理可解释性强，但参数维度高、个体化标定困难。Cannon 2024（Communications Medicine）将猪出血模型与 PROMMTT 多中心临床队列对接，提出基于 ODE 的个性化复苏建议——是当前 THS 数字孪生领域最具临床证据的代表性工作。

**(B) 数据驱动为主：** 基于 EMR/影像/生命体征大数据的统计学习，能融入大型临床数据集，但缺乏生物物理约束、可解释性弱、依赖训练集。大语言模型（LLMs）可基于 EHR 构建预测健康轨迹的虚拟患者表示。

**(C) 机理-数据混合建模：** 包括物理信息神经网络（PINN）、神经常微分方程（Neural ODE）、数据驱动降阶模型、机理引导机器学习等。Tivay 2022 提出 Collective Variational Inference 用于出血复苏个性化建模，是该方向 SOTA；但仍属"离线训练-离线评估"范式，未与可穿戴在线传感器对接、未给出 VVUQ 方案。Sel 2024 在 JAHA 系统提出心血管 DT 临床落地的 V&V&UQ（验证-确认-不确定性量化）框架。

#### 1.2.4 研究现状-差距-本项目应对对照表

| 代表性工作 | 主要贡献 | 与 THS 院前救治的差距 | 本项目应对 |
| ---- | ---- | ---- | ---- |
| Brown 2015, *Sci Transl Med* | 多舱室 ODE 表征虚拟创伤伤员，1 万人队列复现 33 例钝伤血浆 IL-6 | 仅炎症介质，无血流动力学多尺度；不在线个性化 | 把"多系统"扩展到循环-呼吸-神经-内分泌；引入 0D-1D-3D 多分辨率 |
| Jin & Reifman 2023, *IJNMBE* | 战创伤 CR 模型，27 例数据 RMSE 较好 | 纯机理、不可在线个性化、无可穿戴接入 | 引入 PINN 残差学习 + UKF 在线参数辨识 |
| Cannon 2024, *Comm Med* | 猪+PROMMTT 临床数据 ODE 数字孪生，仿真不同复苏方案 | 仅 ODE 全身集总，无脉搏波/局部出血源建模；批处理离线 | 多分辨率 0D-1D-3D + 在线 UKF 数据同化 |
| Tivay 2022, *IEEE TBME* | Collective Variational Inference 个性化推断 | 离线训练，需重训扩展新场景；未对接可穿戴 | 把先验+在线两段式个性化作为 RC3 核心 |
| Sel 2024, *JAHA* | 心血管 DT 原理与 VVUQ 框架综述 | 综述无原型 | 落地为 RC4：VVUQ 工作包 |
| Rubio & Marsden 2024, *CMPB* | 数据驱动 RRI 模型嵌入 0D | 仅冠脉树验证，未用于 THS | 作为 0D-1D-3D 衔接精度补偿模块 |
| DARPA VITAL 2026 | 国家级 CV-DT 方向，机理-数据混合 + 实时仿真干预 | 国内尚无对应布局 | 国内同期同步、面向 THS 急救场景的对应研究 |

**综上**：(1) 现有 THS 数字孪生工作多基于全身 ODE，缺乏多分辨率与在线可穿戴接入；(2) 已有的混合建模 SOTA 仍属离线范式、缺 VVUQ；(3) 国内尚无对接突发事件院前救治的 THS 数字孪生原型。本项目以**机理-数据双驱动 + 多分辨率心血管 + 在线 UKF 个性化 + VVUQ** 形成系统性对策。

## 2. 研究内容、研究目标与拟解决的关键科学问题

### 2.1 研究目标

围绕 THS 院前早期救治的临床需求，研发可在线更新、可前瞻预测、可干预模拟的伤员数字孪生关键技术体系：

- **科学目标：** 揭示大出血扰动下心血管多尺度耦合机制及与呼吸-代谢-神经-内分泌系统交互的数学表征，建立机理-数据双驱动的伤员数字孪生统一建模理论。
- **技术目标：** 突破多分辨率心血管仿真、机理-数据融合多系统建模、稀疏体征下个性化高效参数估计与不确定性量化三项关键技术，形成可便携式终端落地的方法学。
- **应用目标：** 研发自主知识产权的伤员数字孪生原型平台，实现 THS 生理演化趋势预测与复苏方案模拟，为院前救援人员提供"风险-收益"可视化决策辅助。

### 2.2 研究内容

**RC1 多分辨率（0D-1D-3D）可切换心血管模型**
针对大出血伤员血液动力学剧变，开发同时具备实时性的全局 0D 模型与局部高保真 1D/3D 模型，构建 0D-1D-3D 三层耦合架构：0D 始终在线，1D/3D 按事件触发或残差驱动激活，并通过数据驱动 ROM 提供精度补偿。

**RC2 机理-数据双驱动多系统休克模型**
出血休克级联反应跨循环-呼吸-神经-内分泌系统。以机理 ODE 描述血流动力学骨架，以 PINN/Neural ODE 学习神经反射、内分泌调节、毛细血管渗漏、凝血状态等"残差动力学"；机理与 AI 模块通过可微 ODE 求解器端到端联合训练；以物理/生理约束（量纲、守恒、单调性、生理范围）保证可解释性。

**RC3 基于稀疏体征的两阶段在线个性化与不确定性量化**
第一阶段（离线）：Sobol 全局敏感性分析筛选 Top-K（K≈8–15）影响关键体征最大的可辨识参数子集；第二阶段（在线）：仅对 Top-K 参数做 UKF 状态-参数联合估计，输出参数估计值与置信区间；外加 EnKF/CPF 校验。

**RC4 验证、确认与不确定性量化（VVUQ）**
代码验证（与解析解、Pulse、Cannon ODE、openBF 比对）、模型确认（与 PROMMTT/MIMIC-IV/合作医院创伤队列回顾性比对）、不确定性量化（参数 + 数据 + 模型三类 UQ 传播）。VVUQ 既是独立工作包，也贯穿 RC1–RC3 全过程。

### 2.3 拟解决的关键科学问题

| 关键科学问题 (KSP) | 学科基础 | 对应研究内容 | 关键衡量指标 |
| ------ | ------ | ------ | ------ |
| **KSP1**：大出血强非线性扰动下，全身/局部血流动力学多尺度（0D-1D-3D）耦合的统一数学表征与计算可行性 | 计算流体力学 + 集总参数法 + 降阶建模 | RC1 | 关键波形 RMSE；0D 单步推理 ≤1 s |
| **KSP2**：循环-呼吸-神经-内分泌多系统强耦合机制中，机理先验与数据驱动如何在统一损失函数下共优化、保持可解释性 | PINN/Neural ODE + 多任务学习 | RC2 | 与 Cannon/Brown ODE baseline 比较的预测改进；外推一致性指标 |
| **KSP3**：稀疏可穿戴生命体征下，高维生理模型参数的可辨识性、在线估计与不确定性量化 | 数据同化 + 可辨识性分析 + UQ | RC3 + RC4 | Top-K 参数估计置信区间宽度；前瞻性验证 AUC；UQ 校准曲线 |

三个 KSP 之间互为支撑：KSP1 提供"可仿真"的多尺度模型；KSP2 解决"可建模"的多系统耦合；KSP3 解决"可个性化、可信赖"的在线推理。

## 3. 拟采取的研究方案及可行性分析

### 3.1 RC1：多分辨率（0D-1D-3D）可切换心血管模型

**三层架构：**

| 层次 | 模型方法 | 主要变量 | 计算成本 | 在线策略 |
| ---- | ---- | ---- | ---- | ---- |
| 0D 全身闭环（实时引擎） | 改进 Heldt 闭环 + 出血/复苏/神经-内分泌模块 | 压力、流量、容量、神经反射 | 低（毫秒级） | 始终在线 |
| 1D 动脉树（事件触发） | Navier-Stokes，主动脉/颈/锁骨下/髂段 | 压力/流量空间分布、脉搏波传播 | 中 | 残差驱动或事件触发激活 |
| 3D 局部出血源（离线 + 在线降阶） | CFD/流固耦合（腹腔出血、四肢动脉破裂等） | 三维速度场、剪切力 | 高（离线） | 离线 CFD + Kriging/POD-Galerkin 降阶代理在线返还 0D |

**切换/激活准则：**
1. 事件触发：止血带松紧、复苏液体推注、监护点导联切换、ECG 异常事件；
2. 残差驱动：监护波形与 0D 预测残差超阈值（如 MAP 误差 > 10 mmHg 持续 >10 s）；
3. 接口处理：参考 Rubio & Marsden 2024 RRI 数据驱动分叉模型解决 0D 与 1D/3D 衔接处的精度损失。

### 3.2 RC2：机理-数据双驱动的失血性休克多系统建模

**数学结构：**

- **机理层**：心血管 ODE（左/右室、肺循环、体循环、神经反射）+ 呼吸气体交换 ODE（参考 Jin & Reifman 2023）+ 凝血/炎症 ODE（参考 Cannon 2024、Brown 2015）；
- **数据驱动层**：PINN/Neural ODE 拟合毛细血管渗漏系数、激素释放率、组织灌注-代谢偏移等残差动力学；
- **耦合方式**：可微 ODE 求解器（torchdiffeq / DiffEqFlux）使机理与神经网络模块端到端反传；
- **联合损失函数**：
  - 拟合项：对训练队列体征轨迹的 MSE；
  - 物理一致性项：质量/动量守恒、单调性约束、量纲约束；
  - 生理范围项：参数在文献人群分布内的 KL 罚项；
  - 多任务自适应权重（GradNorm / 不确定性加权），平衡循环、呼吸、代谢任务。
- **可解释性校验**：对网络输出做物理量域、守恒、单调性、残差一致性后验校验，未通过则触发回退到纯机理 baseline。

**训练数据：**
1. 合成数据（启动期）：基于 Pulse Engine、Jin & Reifman CR 模型生成 1 万例虚拟伤员；
2. 公开数据：PROMMTT 队列、MIMIC-IV、PhysioNet 创伤数据；
3. 合作医院数据（拟）：经 IRB 审批的回顾性创伤伤员脱敏数据。

### 3.3 RC3：基于稀疏体征的两阶段个性化高效参数估计

**第一阶段：离线敏感性 + 可辨识性分析**
- 用 Sobol 全局敏感性指数对 RC2 模型中 100+ 参数做排序；
- 按一阶/总指数选 Top-K（K ≈ 8–15）影响 MAP/HR/ETCO₂/SpO₂ 最显著的参数子集；
- 剩余参数固定为基于人口学特征（性别、年龄、体重、损伤部位）的人群均值/贝叶斯先验。

**第二阶段：在线 UKF 状态-参数联合估计**
- 仅对 Top-K 参数 + 关键状态变量做联合估计；
- 使用正则化协方差更新（Joseph form）保证正定；约束化 σ 点投影把 σ 点限制在生理可行区间；
- 输出参数估计值 + 95% 置信区间；
- 外层叠加 EnKF/条件粒子滤波（CPF）做交叉校验，UKF 失败时自动回退；
- 给出"风险-收益"可视化：复苏方案 A/B 在 30/60/120 min 后的伤情曲线 + 不确定带。

**个性化建模流程：**
人口学初始化 → 稳态运行 → 接入可穿戴体征流（采样率：心电 250 Hz、SpO₂ 1 Hz、NIBP 0.05 Hz、ETCO₂ 25 Hz）→ Sobol Top-K 启动 → UKF 在线辨识 → CPF 校验 → 输出个性化预测 + 不确定度。

### 3.4 RC4：验证、确认与不确定性量化（VVUQ）

| 层次 | 内容 | 数据/工具 | 量化指标 |
| ---- | ---- | ---- | ---- |
| 代码验证（Verification） | 数值收敛性、与解析解/已发表模型一致性 | 解析 Windkessel 解、Pulse Engine、Cannon ODE 公开代码、openBF | 收敛阶；MRE < 5% |
| 模型确认（Validation） | 与回顾性临床数据吻合度 | PROMMTT、MIMIC-IV、合作医院 IRB 队列；健康志愿者监测样本 | MAP/HR/ETCO₂/SpO₂ 的 RMSE、相关系数；事件 AUROC、Brier score |
| 不确定性量化（UQ） | 参数、数据、模型不确定度分解与传播 | Sobol、贝叶斯神经网络、Ensemble | 95% PI 覆盖率；Calibration ECE |

### 3.5 数据来源与伦理合规

1. **公开数据**：MIMIC-IV、PROMMTT、PhysioNet 急救数据集；遵循 DUA 与 PhysioNet credentialing 协议。
2. **合成数据**：基于 Pulse Engine、Jin & Reifman CR 模型生成虚拟伤员队列（≥1 万样本），无伦理风险。
3. **合作医院**（建议补充：__________ 医院/解放军总医院/新疆医科大学第一附属医院）回顾性脱敏创伤数据，已与/拟向单位 IRB 申请伦理审查，按《人类遗传资源管理条例》与《数据安全法》规范处置。
4. **小样本健康志愿者监测**通过校 IRB 审批，知情同意书完备，不涉及对患者的干预。
5. **数据存储**：脱敏后存入科研内网；模型仅在数据所在域内训练，不外传原始数据。

### 3.6 关键技术与可行性

- **多分辨率耦合**：Rubio & Marsden 2024 RRI 已在 0–5500 雷诺数下完成多分叉验证，技术成熟；本项目延伸至出血扰动场景；
- **机理-数据融合**：torchdiffeq、Diffrax 等可微 ODE 工具链成熟，PINN 在 ESICM/JAHA 综述均已被列为 DT 落地路径；
- **UKF + Sobol**：申请人前期已掌握；
- **数据可获得性**：Cannon 2024 已公开 ODE 代码与部分 PROMMTT 拟合脚本，可作为机理 baseline 直接复现；
- **算力**：实验室拥有 GPU 工作站；离线 3D CFD 拟在校超算上完成。

## 6. 研究风险与应对

| 风险点 | 影响 | 应对策略 |
| ---- | ---- | ---- |
| 创伤伤员真实数据可及性差、IRB 审批周期长 | RC2/RC4 数据不足 | 三层数据策略（合成→公开→医院）；前两层 6 个月内启动；预留与多家医院并行申请的备份 |
| Sobol+UKF 后仍存在不可辨识参数 | 在线辨识发散 | 引入贝叶斯优化做 K 值自适应；UKF 失败回退 EnKF/CPF；预设保守先验 |
| PINN/Neural ODE 训练不稳定 | RC2 进度滞后 | 预训练-微调；以 Cannon 2024 ODE 作为热启动；保留纯机理 baseline 兜底 |
| 0D-1D-3D 耦合数值不稳定 | 仿真发散 | 隐式时间步进；RRI 分叉模型补偿；多平台（FEniCS/openBF/SimVascular）交叉验证 |
| 便携式终端算力受限 | 现场演示受限 | 0D 引擎 C++ 实现 + ONNX 量化；3D 仅离线运行；混合云-边架构兜底 |
| 临床合作单位变更 | 数据计划受影响 | 同时与 ≥2 家三甲医院谈合作，并锁定 PROMMTT/MIMIC-IV 公开队列底线 |
