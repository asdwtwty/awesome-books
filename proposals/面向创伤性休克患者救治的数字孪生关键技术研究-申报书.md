## 一、项目摘要

创伤性休克是战创伤、重大事故和突发公共安全事件中最主要的早期致死原因，其中**大失血（hemorrhagic shock）**、**气道梗阻（airway obstruction）**和**张力性气胸（tension pneumothorax）**所致"可预防性死亡"占战场伤亡的 60%–90%。然而在院前急救、战场前接和灾难现场，救治窗口短、生命体征演变快、评估高度依赖经验，缺乏面向个体的、可前瞻预测伤情演变的智能决策支撑工具。

本项目以构建**面向创伤性休克救治的患者级数字孪生体（Patient-Specific Digital Twin, PS-DT）**为目标，以"大失血—气道梗阻—张力性气胸"三类高致死性伤情为核心研究场景，融合**机理模型（知识驱动）与机器学习模型（数据驱动）**、**几何（结构）模型与生理（功能）模型**，构建多尺度耦合的虚拟伤员；通过**可穿戴/便携式多模态传感器**实时采集伤员体征，建立**在线个性化参数估计与同化**机制，使数字孪生与真实伤员状态保持一致；并以此为基础**预测未来 5–60 分钟生命体征演变趋势**，输出休克分级、失代偿预警与处置建议，为战创伤救治、院前急救与重症转运提供新一代智能决策支持。

## 二、立项依据

### 2.1 研究背景与意义

#### 2.1.1 创伤性休克救治面临的严峻挑战

创伤是 45 岁以下人群死亡的首位原因，其中创伤性休克在伤后 1 小时（"白金 10 分钟、黄金 1 小时"）的死亡率最高。美军战伤数据分析（OEF/OIF）表明：

- **大失血**：占战场可预防性死亡的 60% 以上，躯干非压迫性出血尤为致命；
- **张力性气胸**：占可预防死亡的 13%–33%，从代偿到循环崩溃常仅有数分钟；
- **气道梗阻**：占可预防死亡的 1%–8%，颌面伤、烧伤、误吸等导致快速窒息。

上述三类伤情**起病急、演变快、临床表现非特异**，传统基于离散时间点测量与临床经验的判别方式存在**滞后性**和**主观性**，亟需建立**前瞻性、个性化、可解释**的智能预测体系。

#### 2.1.2 数字孪生在医疗领域的战略价值

数字孪生（Digital Twin）通过对物理对象建立"实时同步、双向交互"的虚拟镜像，已在航空、能源、智能制造领域取得突破。近年来 NIH、DARPA、美军 TATRC、欧盟 EDITH 等相继布局"医疗数字孪生（Medical Digital Twin, MDT）"，并将其列入未来精准医学和军事医学核心技术。我国《"十四五"医疗装备产业发展规划》《国家中长期科技发展规划纲要》也明确支持"医工交叉""智能急救""战创伤救治"等方向。

围绕创伤性休克救治构建患者级数字孪生，具有重要的**科学价值**（多尺度生理建模与数据同化的方法学突破）、**军事价值**（提升战场前接救治能力）和**民用价值**（赋能院前急救、5G 救护车、灾难医学）。

### 2.2 国内外研究现状

**(1) 创伤生理机理建模**：以 ECCE/HumMod、PhysioLab、BioGears 为代表的全身生理模型可模拟失血、补液、机械通气等过程，但参数高度通用，难以反映个体差异；对张力性气胸、气道梗阻等结构性病变缺乏几何建模。

**(2) 数据驱动的生命体征预测**：基于 MIMIC-IV、eICU、TraumaRegister DGU 等数据集，已涌现 LSTM、Transformer、TCN 等模型预测休克指数、低血压事件、ARDS 风险。其优势是端到端学习能力强，但**外推能力差、可解释性弱、对分布外伤情泛化困难**，难以单独应用于战创伤场景。

**(3) 几何—生理耦合建模**：在心血管（如 SimVascular）、肺通气（如 lung4D）等领域已出现影像驱动的 0D/1D/3D 多尺度模型，但面向创伤、可在前接环境运行的轻量化耦合模型尚处空白。

**(4) 可穿戴体征监测**：心电贴片、PPG 手环、无袖带血压、贴片式呼吸监测、便携超声等设备已成熟，但**多模态融合、运动伪迹抑制、低带宽下的状态同化**仍是公认难题。

**总体而言**：现有研究在"机理 vs 数据""结构 vs 功能""离线 vs 在线""通用 vs 个性化"四对矛盾上尚未形成统一框架，**面向创伤性休克的患者级数字孪生关键技术体系尚未建立**，是本项目主攻的方向。

### 2.3 研究价值

- **理论价值**：建立融合知识与数据的多尺度生理数字孪生方法学，发展可在线辨识的混合建模理论；
- **技术价值**：突破可穿戴稀疏体征下的高保真状态估计、伤情演变前瞻预测等关键技术；
- **应用价值**：形成可在战创伤前接、院前急救、急诊抢救室、ICU 转运等场景部署的原型系统，显著提升早期救治决策水平。

## 三、研究目标、研究内容与拟解决的关键科学问题

### 3.1 总体研究目标

围绕**大失血、气道梗阻、张力性气胸**三类典型伤情，构建一套**机理—数据双驱动、几何—生理双耦合、监测—估计—预测三闭环**的创伤性休克患者数字孪生关键技术体系，并形成可演示验证的原型系统：

- 数字孪生可在 ≤ 1 秒延迟内反映真实伤员状态；
- 关键体征（HR、RR、SpO₂、SBP/MAP、EtCO₂、SI 等）**未来 30 分钟预测平均误差 < 10%**，**休克失代偿预警提前量 ≥ 15 分钟**，**ROC-AUC ≥ 0.90**；
- 在仿真伤员数据集和回顾性临床/战创伤数据集上完成系统验证。

### 3.2 主要研究内容

#### 内容一：面向三类典型伤情的多尺度机理—几何耦合建模

构建覆盖"系统—器官—组织"三个尺度的虚拟伤员：

1. **几何（结构）模型**：基于人体解剖图谱与代表性 CT/MRI 影像，重建胸腔、肺、气道、心脏、大血管和典型出血部位的 3D 几何，建立可参数化的"伤情几何变形算子"（出血腔扩张、肺塌陷、气道狭窄）；
2. **生理（功能）模型**：建立 0D/1D 心血管—呼吸—代谢—神经体液耦合机理模型（含压力—容积—阻力—顺应性集总参数、压力反射、RAAS、代谢酸中毒等），刻画**失血—代偿—失代偿—不可逆**演变规律；
3. **几何—生理耦合**：将几何参数（出血量、胸腔积气体积、气道有效直径等）作为生理模型的时变参数注入，实现结构变化对功能的实时驱动。

#### 内容二：机理（知识驱动）与机器学习（数据驱动）融合建模

针对纯机理模型"个体适配差"和纯数据模型"外推与可解释差"的问题，研究以下混合建模范式：

1. **物理信息神经网络（PINN）**：将心血管—呼吸控制方程作为软约束注入神经网络损失函数；
2. **神经常微分方程（Neural ODE）/ UDE（Universal Differential Equation）**：以机理 ODE 为骨架，由神经网络学习残差项，弥补未建模动力学；
3. **机理模型作为数据增广器**：用机理模型生成大规模"虚拟伤员"轨迹用于预训练，再以真实数据微调；
4. **可解释性后端**：基于 SHAP / 反事实推理 / 机理回溯，提供"哪一项体征驱动了预警"的解释。

#### 内容三：基于可穿戴多模态传感的实时状态感知

1. **多模态传感方案**：心电贴片（HR/HRV/心律）、PPG（SpO₂、PPG-PTT 推算无袖带血压）、贴片式胸阻抗与加速度计（呼吸频率/幅度/不对称运动）、便携 EtCO₂、便携掌超（FAST/eFAST 半自动判读，可选）；
2. **运动伪迹与缺失数据处理**：研究基于自监督表征学习与因子图融合的伪迹抑制与缺失补全；
3. **边缘—云协同**：构建边缘端轻量推理（INT8/蒸馏模型）+ 云端高保真孪生的双层架构，适应战场低带宽环境（< 64 kbps 可工作）。

#### 内容四：在线个性化参数估计与数据同化

1. **参数辨识**：采用集合卡尔曼滤波（EnKF）、无迹卡尔曼（UKF）、变分贝叶斯、近似贝叶斯计算（ABC-SMC）等方法，从稀疏观测中辨识个体化参数（血容量、外周阻力、肺顺应性、气道阻力、胸腔顺应性、压力反射增益等）；
2. **可辨识性分析与降阶**：基于 Sobol 灵敏度与 Fisher 信息矩阵筛选可辨识参数，构建低维 ROM；
3. **数据—模型同化**：建立"传感观测—孪生状态"的递归同化机制，使孪生体在线收敛到真实伤员；
4. **不确定性量化（UQ）**：输出预测区间而非点估计，支持"风险感知"的临床决策。

#### 内容五：未来生命体征演变趋势预测与失代偿预警

1. **多步预测**：在已估计的个性化模型上滚动外推未来 5/15/30/60 分钟的 HR、RR、MAP、SpO₂、EtCO₂、SI、ETCO₂/SpO₂ 等轨迹；
2. **休克失代偿预警**：建立"代偿—早期失代偿—失代偿—濒危"四级状态机，结合 CRI、SI、PI、ΔMAP 等指标输出预警；
3. **干预效果模拟**："What-if" 模式：模拟胸腔穿刺减压、环甲膜切开、止血带、限制性液体复苏、TXA 等处置后的孪生轨迹，辅助救治决策；
4. **决策支持接口**：以战术医疗员/急救员为用户，输出简洁可操作的处置建议清单。

#### 内容六：原型系统集成、验证与示范

完成"可穿戴前端 + 边缘网关 + 云端孪生 + 临床/战训终端"的原型系统集成；
在**仿真伤员数据集**（基于机理模型生成的多伤情虚拟伤员队列）、**回顾性临床数据集**（MIMIC-IV、eICU、合作医院创伤数据库）以及**模拟训练数据**（高仿真模拟人 TraumaMan / METIman / 仿生气道模型）三类场景下完成系统验证。

### 3.3 拟解决的关键科学问题

- **KP1**：稀疏可穿戴体征下，结构—功能耦合多尺度生理模型的**可辨识性边界**与**最小观测集**？
- **KP2**：机理模型先验与数据驱动模型如何在**有限标注、分布漂移、伤情罕见**条件下实现**鲁棒、可解释的混合建模**？
- **KP3**：在伤情快速演变下，如何实现**实时数据同化**与**不确定性可控的前瞻预测**？

### 4.1 总体技术路线

│  │ 胸腔/肺/血管 │    │心血管/呼吸/代谢│    │ PINN / Neural ODE   │ │
│  └──────┬───────┘    └──────┬───────┘    └──────────┬───────────┘ │

│                            ▲                                       │
└────────────────────────────┼───────────────────────────────────────┘
                             │  状态/参数估计 (EnKF/UKF/VB/ABC)
                             │

   │           可穿戴/便携多模态传感前端                       │

### 4.2 各研究内容的具体技术方案

#### 方案 1：三类伤情的机理—几何耦合模型

- **大失血**：在 0D 闭环心血管模型基础上，引入"出血源—血容量动态—压力反射—外周血管收缩—代谢酸中毒"耦合方程；几何上以可参数化出血腔表征非压迫性出血部位（腹腔、盆腔、胸腔、长骨）；
- **气道梗阻**：建立可变截面的中央/上气道 1D 流动模型（基于 Navier–Stokes 简化 + 局部 Starling 阻力子），将梗阻位置、程度、持续时间作为时变参数；耦合至呼吸肌—肺—气体交换 0D 模型；
- **张力性气胸**：建立胸腔—肺—心血管耦合模型，以胸膜腔气体体积为状态变量，描述其对**肺顺应性下降、纵隔移位、静脉回流减少、心输出量下降**的级联效应；几何端基于 CT 模板的可变形胸腔模型生成不同程度气胸的影像/结构表征。

#### 方案 2：混合建模实现路径

- **骨架方程 + 神经残差**：`dx/dt = f_mech(x, θ) + NN_φ(x, u)`，其中 `f_mech` 为机理项，`NN_φ` 学习残差；通过对偶损失（数据拟合 + 物理一致性 + 参数先验）联合训练；
- **机理预训练 + 真实微调**：先用机理模型生成 10⁵ 量级虚拟伤员轨迹预训练 Transformer 时序模型，再用真实数据 LoRA 微调；
- **不确定性传递**：MC Dropout / 深度集成 / 共形预测（Conformal Prediction）输出可校准置信区间。

#### 方案 3：可穿戴感知与边缘计算

- **传感器选型**：胸贴式 ECG（采样率 ≥ 250 Hz）+ 腕式/指端 PPG + 胸阻抗呼吸贴片 + 鼻氧管/面罩集成 EtCO₂ + 三轴加速度（姿态/搬运伪迹）；
- **运动伪迹抑制**：基于自监督对比学习（SimCLR-style）训练体征鲁棒表征；
- **边缘端**：ARM Cortex-A 级 SoC + 轻量化 ROM（< 50 MB）模型，10 Hz 状态更新；
- **通信**：MQTT over 5G/4G/卫星，支持低带宽降级模式（仅上传状态摘要 + 关键事件）。

#### 方案 4：在线参数估计

- 采用 **EnKF + UKF 双滤波**：EnKF 处理高维状态、UKF 处理强非线性参数；
- **可辨识性分析**：先通过 Sobol/Morris 全局灵敏度筛除不可辨识参数，再以 Fisher 信息约束辨识范围；
- **冷启动问题**：使用人群级先验分布（基于年龄、性别、体重、伤情类型）进行 ABC-SMC 初始化。

#### 方案 5：预测与决策支持

- **多步滚动预测**：以同化后的孪生作为初值，机理 + 残差 NN 联合外推；
- **预警判定**：基于状态轨迹 + 阈值 + 学习型分类器，输出四级预警；
- **What-if 模拟**：将处置（穿刺减压、止血带、气道开放、补液速率）作为模型输入扰动，比较干预/不干预的预测分布差异。

#### 方案 6：验证与示范

- **仿真验证**：构造涵盖年龄、合并伤、伤情严重度的 1000 例虚拟伤员队列；
- **回顾性数据验证**：与合作医院创伤中心、军队卫勤单位合作，基于伦理审查后的脱敏数据回顾验证；
- **模拟训练验证**：在高仿真模拟人/模拟训练课目中实测原型系统；
- **指标体系**：MAE/MAPE/RMSE、Pinball Loss（区间预测）、AUC/AUPRC（预警）、Lead Time（提前量）、临床可用性 SUS 量表。

### 4.3 可行性分析

- **理论可行**：心血管—呼吸 0D/1D 建模、PINN/Neural ODE、EnKF 数据同化均有成熟方法学基础；
- **数据可行**：MIMIC-IV、eICU、AmsterdamUMCdb 公开重症数据集已可获取；与 [拟合作医院/卫勤单位] 已建立创伤数据共享机制（拟签署合作协议）；
- **硬件可行**：所选可穿戴传感器均为成熟产品（ECG 贴片、PPG 手环、EtCO₂ 模块），无需新研硬件；
- **算力可行**：边缘端模型已通过初步压缩验证可在 SoC 上 10 Hz 推理，云端使用现有 GPU 集群；
- **团队可行**：申报团队由生物医学工程、临床急诊/创伤外科、机器学习、嵌入式系统四方向骨干组成，前期已有 [机理模型原型 / 体征预测论文 / 可穿戴前端] 等积累。

## 五、本项目的特色与创新之处

1. **场景创新**：首次面向"大失血—气道梗阻—张力性气胸"三类**战创伤可预防性死亡核心伤情**构建患者级数字孪生体系；
2. **方法创新**：提出"**机理 ⊕ 数据 ⊕ 几何 ⊕ 生理**"四元融合的混合建模框架，机理作为骨架与正则、数据负责弥补未建模动力学、几何驱动结构性病变的功能后果；
3. **范式创新**：构建"**可穿戴稀疏感知 → 在线个性化参数估计 → 前瞻预测 → 干预模拟**"的闭环数字孪生范式，区别于传统"事后分析型"AI；
4. **应用创新**：贯通**战场前接—院前急救—转运 ICU**的全链条急救数字孪生原型，可直接服务于卫勤、120、5G 救护车与急诊抢救室。

## 八、组织实施与风险控制

### 8.1 组织架构

- **课题一组（机理建模）**：负责几何—生理耦合模型；
- **课题二组（机器学习）**：负责混合建模与预测预警；
- **课题三组（可穿戴与边缘）**：负责传感前端与边缘计算；
- **课题四组（临床验证）**：负责数据治理、临床验证与示范应用；
- **质量与合规组**：负责伦理、数据安全、模型评测。

### 8.2 主要风险与对策

| 风险类别 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 数据风险 | 真实战创伤数据稀缺、标注成本高 | 以机理模型生成虚拟伤员数据预训练 + 回顾性临床数据微调 + 模拟训练数据补充 |
| 模型风险 | 在线参数估计在稀疏观测下不可辨识 | 灵敏度筛选、降阶模型、人群级先验、Bayesian UQ |
| 工程风险 | 边缘算力受限、低带宽通信不稳 | 模型蒸馏/量化、状态摘要降级上行、断网本地决策 |
| 合规风险 | 数据隐私与伦理 | 全程 IRB 审查、联邦学习/差分隐私、严格脱敏 |
| 转化风险 | 医工跨学科协作不畅 | 设立联合 PI 与定期临床—工程评审会，建立闭环反馈 |
