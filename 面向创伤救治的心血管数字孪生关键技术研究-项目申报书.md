## 一、项目摘要

战创伤是和平时期与战时致死致残的首要原因，其中心血管系统失代偿（失血性休克、心肌损伤、循环衰竭等）是导致"可预防性死亡"的主要病理机制。受限于黄金救治时间短、伤情演变快、个体差异大、前沿救治资源匮乏等现实约束，传统经验式救治难以及时识别隐匿性出血和循环代偿耗竭等关键节点。

本项目面向战创伤救治的现实需求，研究"机理 + 数据"双驱动、"结构 + 功能"一体化的**心血管数字孪生关键技术**，通过可穿戴设备实时采集伤员的多模态生理信号，反演与同化个性化模型参数，构建可在线推演的伤员心血管虚拟副本，从而实现伤情演变趋势预测、循环失代偿早期预警、救治决策辅助。项目预期在多尺度心血管建模、个性化参数辨识、模型–数据同化、轻量化在线推断等方向取得突破，形成可在便携式终端部署的伤员心血管数字孪生原型系统。

**关键词**：数字孪生；心血管系统；战创伤救治；机理–数据融合建模；个性化参数辨识；可穿戴监测；伤情预测

## 二、立项依据

### 2.1 研究背景与意义

1. **战创伤救治的现实痛点**
   - 战创伤死亡中约 30%–40% 属于"可预防性死亡"，其中失血性休克占首位。
   - 伤后 1 小时（黄金时间）和前 10 分钟（白金十分钟）内的处置质量直接决定结局。
   - 在远前方、海上、空中、灾害现场等场景下，缺乏专科医生与影像/检验设备，救治高度依赖单兵传感与远程指挥。

2. **传统监测手段的局限**
   - 心率、血压、SpO₂ 等单一阈值告警滞后于真实病理生理变化（如年轻伤员血压代偿期可长达数十分钟）。
   - 不同伤员的循环储备、代偿能力、合并症差异巨大，"一刀切"阈值带来高漏报率与高误报率。
   - 缺乏对未来 5–60 分钟伤情演变的**前瞻性**评估能力。

3. **数字孪生为破局提供了新范式**
   - 数字孪生通过"物理实体–虚拟模型–数据闭环"实现对个体生理状态的高保真映射与超前推演。
   - 心血管系统具有相对清晰的物理机制（流体力学、弹性力学、控制反馈），适合机理建模。
   - 可穿戴/便携设备（PPG、ECG、ICG、NIRS、IMU 等）已具备战场级部署条件，可为孪生体提供持续数据流。

### 2.2 国内外研究现状

- **机理模型方向**：0D 集中参数模型（Windkessel、闭环 CVS 模型）、1D 一维波传播模型、3D CFD/FSI 模型已较成熟，但参数个性化和实时求解仍是瓶颈。
- **数据驱动方向**：基于深度学习的 ABP 重建、休克指数预测、出血量估计等取得显著进展，但模型可解释性差、对分布外（OOD）伤情外推能力弱。
- **数字孪生方向**：心脏电生理孪生（如 SIEMENS、Dassault Living Heart）、个性化主动脉孪生已在临床手术规划中使用，但**面向创伤动态演变的孪生体仍处空白**。
- **关键差距**：① 缺乏机理与数据深度融合的统一建模框架；② 缺乏从可穿戴稀疏信号反演高维心血管参数的高效方法；③ 缺乏可在便携终端实时运行的伤情外推引擎。

### 2.3 科学问题凝练

- **关键科学问题 1**：如何构建机理可解释、数据可校正的多尺度心血管孪生模型？
- **关键科学问题 2**：如何在可穿戴稀疏、噪声、间歇观测条件下，对高维个性化参数进行可辨识、可追踪的在线估计？
- **关键科学问题 3**：如何在伤情非平稳演变下保持孪生体的预测精度与泛化能力？

## 三、研究目标与研究内容

### 3.1 总体目标

构建一套**面向创伤救治的心血管数字孪生理论与技术体系**，形成"几何–生理一体、机理–数据共融、感知–反演–预测闭环"的伤员心血管孪生原型系统，实现：

- 心血管几何与功能模型的个体化构建时间 ≤ 5 分钟；
- 基于可穿戴信号的关键参数在线辨识误差 ≤ 15%；
- 失血性休克、循环失代偿等关键事件的 15–30 分钟超前预警 AUC ≥ 0.90；
- 在便携式 GPU/NPU 终端实现 ≥ 10× 实时推演速率。

### 3.2 研究内容

项目设置 5 项相互衔接的研究内容，整体技术路线见 §4。

#### 研究内容 1：心血管几何（结构）模型的快速个性化构建

- 基于伤员入伍/入院体检的有限影像（CT/MRI/超声）或人口统计学先验（性别、年龄、身高、体重、BSA），通过**统计形状模型（SSM）+ 扩散生成模型**生成个性化心腔–大血管几何。
- 研究形状先验与稀疏测量的贝叶斯融合方法，实现"少样本–高保真"几何重建。
- 输出：个性化 3D 心脏 + 主动脉 + 主要分支血管网格，作为机理模型的几何载体。

#### 研究内容 2：机理驱动的多尺度心血管生理（功能）模型

- 0D 闭环模型：包含左/右心室时变弹性、肺循环、体循环、压力反射（baroreflex）、化学反射、自主神经调节。
- 1D 模型：覆盖主动脉–大动脉–微循环的脉搏波传播，用于解释 PPG/PWV/PTT 等可穿戴信号成因。
- 3D 模型（按需调用）：在严重伤情或决策需要时，对关键解剖部位进行 CFD/FSI 高保真求解。
- 创伤特征模块：失血源项、组织损伤致儿茶酚胺释放、代偿性血管收缩、心肌挫伤导致的收缩力下降等病理参数化模型。
- 多尺度耦合：0D ↔ 1D ↔ 3D 的边界条件双向耦合，统一在隐式时间推进框架下求解。

#### 研究内容 3：机理（知识驱动）与机器学习（数据驱动）融合建模

- **物理嵌入神经网络（PINN/Physics-Informed Operator Network）**：将守恒律、Windkessel 方程作为软约束嵌入损失函数，提升小样本泛化能力。
- **混合代理模型（Hybrid Surrogate）**：以 0D/1D 解为骨架，残差用神经算子（FNO、DeepONet）拟合，兼顾速度与精度，达到机理模型的精度、数据模型的速度。
- **不确定性量化**：基于贝叶斯神经网络与共形预测，对孪生体输出给出可信区间，避免"过自信"误导救治决策。
- **闭环可微分仿真**：整套模型可微，便于梯度反向传播至个性化参数。

#### 研究内容 4：可穿戴感知与个性化参数在线辨识

- **多模态可穿戴感知层**：ECG、PPG（多波长）、ICG/BioZ、NIRS（脑/肌氧）、皮温、IMU（体位、呼吸）、可选无创血压袖带。
- **信号质量评估与缺失填补**：在战场强运动伪影、断续通信下保证可用观测。
- **数据同化引擎**：
  - 集合卡尔曼滤波（EnKF）/ 无迹卡尔曼滤波（UKF）用于快速时变状态估计；
  - 变分同化（4D-Var）+ 自动微分用于慢变结构参数（血管顺应性、心肌收缩力等）辨识；
  - 神经替代器辅助的 SMC（粒子滤波）用于非高斯、非线性场景。
- **可辨识性分析**：基于 Fisher 信息矩阵与全局灵敏度（Sobol 指数）评估在给定观测下哪些参数"可被看见"，避免无效辨识。

#### 研究内容 5：伤情演变预测与救治决策辅助

- 将辨识完毕的孪生体作为**伤员级模拟器**，超前推演 5/15/30/60 分钟的循环轨迹。
- 在孪生体上对候选干预方案（补液、止血带、血管活性药、压迫止血、后送优先级）进行**反事实仿真**，输出预期收益与风险。
- 基于强化学习/贝叶斯优化的救治建议生成模块，并提供机理可解释的"为什么"说明，供医务人员参考。
- 形成**伤情演变指数（Trauma Trajectory Index, TTI）**：综合反映"当前严重度 + 未来恶化速率 + 干预敏感度"的复合指标。

### 4.2 关键算法设计

#### 4.2.1 几何模型构建（研究内容 1）

- 训练集：公开心血管影像库（如 MMWHS、VMR）+ 自建伤员体检影像。
- 方法：PCA-SSM 提取低维形状空间 → 条件扩散模型 `p(geometry | age, sex, BMI, sparse_landmarks)` 生成个体几何 → 网格质量后处理（重网格、边界层）。

#### 4.2.2 机理–数据融合模型（研究内容 2、3）

设系统状态为 $x(t)$，参数为 $\theta$，机理动力学为：

$$\dot{x}(t) = f_{\text{phys}}(x, \theta, u) + g_\phi(x, \theta, u; w)$$

其中 $f_{\text{phys}}$ 为机理项（守恒律、Windkessel、心室弹性方程等），$g_\phi$ 为以神经网络参数化的残差校正项，$w$ 为可学习权重。训练损失：

$$\mathcal{L} = \underbrace{\|x_{\text{pred}} - x_{\text{obs}}\|^2}_{\text{数据项}} + \lambda_1 \underbrace{\|\mathcal{R}_{\text{phys}}(x_{\text{pred}})\|^2}_{\text{物理残差}} + \lambda_2 \underbrace{\Omega(w)}_{\text{正则}}$$

通过逐步退火 $\lambda_1$ 实现"先机理后数据"的稳定训练。

#### 4.2.3 个性化参数在线辨识（研究内容 4）

针对慢变参数 $\theta_s$（顺应性、外周阻力、收缩力）与快变状态 $x$ 采用**双时间尺度估计**：

- 快尺度（秒级）：UKF 跟踪 $x$；
- 慢尺度（分钟级）：基于一段窗口数据，使用可微分仿真 + L-BFGS 优化 $\theta_s$；
- 触发式重辨识：当残差超过阈值（提示伤情阶跃，如新发出血）时强制更新 $\theta_s$。

可辨识性分析：

$$\mathcal{F}_{ij}(\theta) = \mathbb{E}\left[\frac{\partial \log p(y|\theta)}{\partial \theta_i}\frac{\partial \log p(y|\theta)}{\partial \theta_j}\right]$$

仅辨识 $\mathcal{F}$ 主导特征方向上的参数，其余以先验固定。

#### 4.2.4 伤情外推与决策（研究内容 5）

- 在 $t$ 时刻冻结当前孪生体参数 $\hat\theta(t)$，向前积分至 $t+\Delta t$ 得到预测轨迹 $\{\hat x(\tau)\}_{\tau=t}^{t+\Delta t}$。
- 对每个候选干预 $a_k$：以干预后参数/输入扰动 $(\theta', u')$ 重新积分，得到反事实轨迹。
- 决策准则：

$$a^* = \arg\max_{a_k}\; \mathbb{E}\big[U(\hat x_{a_k}(t+\Delta t))\big] - \beta\, \text{Risk}(a_k)$$

其中 $U$ 为综合效用（MAP 维持、组织灌注、SaO₂ 等），$\text{Risk}$ 来自共形预测得到的不确定性区间。

### 4.3 验证方案

1. **体外验证**：搭建心血管模拟回路（mock circulation loop），人为制造失血、阻力变化等扰动，验证孪生体的辨识与预测精度。
2. **动物实验**：与军事医学/创伤医学单位合作，在猪失血性休克模型上验证孪生体对 MAP、CO、SVR 演变的预测性能（单位主导，本项目提供算法）。
3. **回顾性临床数据**：基于 MIMIC-IV、eICU、自建创伤 ICU 数据进行离线评估。
4. **前瞻性可穿戴小队测试**：选拔志愿者在标准化运动 + 体位倾斜（HUT）、下体负压（LBNP）等模拟容量丢失场景下采集数据，作为类休克行为的安全替代验证。
5. **指标体系**：参数辨识 NRMSE、轨迹预测 MAE/dCRPS、事件预警 AUC/AUPRC/Lead-time、决策模拟 Regret。

## 五、创新点

1. **建模范式创新**：提出"机理–数据双驱动 + 几何–生理一体化"的心血管孪生统一框架，机理项保证可解释与外推能力，数据项补足建模残差，几何项约束生理项，三者形成内在一致的可微闭环。
2. **辨识方法创新**：面向可穿戴稀疏观测，提出"双时间尺度同化 + 可辨识性引导"的个性化参数估计方法，解决高维参数在低维观测下的不可辨识难题。
3. **应用场景创新**：率先把数字孪生从择期手术规划推进到**动态、非平稳、强不确定性的战创伤救治**场景，提出 Trauma Trajectory Index 与反事实救治推演范式。
4. **工程实现创新**：通过神经算子代理 + 模型蒸馏，实现孪生体在便携终端的 ≥10× 实时推演，使数字孪生具备"伴随单兵"的部署能力。

## 九、风险分析与应对

| 风险 | 影响 | 应对措施 |
|---|---|---|
| 战创伤真实数据稀缺 | 模型训练受限 | 以模拟回路 + 动物模型 + LBNP 等替代场景为主；引入合成数据增广；多中心联邦学习 |
| 可穿戴信号在战场环境下质量差 | 同化失稳 | 信号质量评估门控；缺失–鲁棒同化；机理模型作为兜底先验 |
| 高维参数不可辨识 | 个性化失败 | 可辨识性分析驱动的参数子集选择；分层先验；多任务共享表征 |
| 算力受限 | 终端无法实时 | 多保真度建模 + 神经算子代理 + 蒸馏量化；按需上云 |
| 伦理与监管风险 | 临床落地受阻 | 提前介入伦理审查；定位为"决策辅助"而非"自主决策"；全流程可追溯 |

## 十一、伦理与数据合规

- 所有人体数据采集均通过医学伦理委员会审批，签署知情同意。
- 数据传输与存储采用端到端加密；个人标识与生理数据分库；提供数据可销毁机制。
- 模型输出明确标注"仅供医务人员参考"，并附不确定性区间，避免越权决策。
- 涉军/涉密数据按相应保密制度执行。

## 十二、结语

本项目以战创伤救治这一国家与社会重大需求为牵引，将数字孪生、机理建模、机器学习、可穿戴感知等前沿技术深度融合，致力于为伤员构建一个"懂他、看远、可推演"的心血管虚拟副本。期望通过 3 年攻关，形成一套理论自洽、工程可行、可向临床与战场延伸的关键技术体系，为降低可预防性死亡、提升精准救治能力作出实质性贡献。
